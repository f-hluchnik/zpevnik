\hyt{tomamemladez}
\song{To máme mládež}

\vers{1}{
Jen \chord{E}zpívají si rock a zpívají si rock,\\
no já vám povím, milá paní, \chord{E\7}škoda pro ně škol.
}

\refrain{
To máme \chord{A}mládež, to máme \chord{E}mládež!\\
Doma \chord{H\7}řezali je málo, \chord{A\7}jak se zdálo, \chord{H\7}kam to vede se \chord{E}ptám?
}

\vers{2}{
To včera jeden kluk mě sednout nepustil,\\
a vymlouval se na frakturu, to je jejich styl.
} \refsm{}

\vers{3}{
Ty holky jen tak tak se vejdou do svejch džín,\\
a tyhle jejich mini mini, já jsem z toho hin.
} \refsm{}

\vers{4}{
A všechno chtějí hned a všechno vědí líp,\\
a když jim člověk moudře radí, mají na to vtip.
} \refsm{}

\vers{5}{
No já vám povídám, my byli jinačí,\\
i když se o nás taky neslo naší pavlačí.
} \refsm{}
