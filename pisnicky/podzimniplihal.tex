\sub{Karel Plíhal}

\note{capo 4}

\vers{a}{
\chord{G}Podzimní obloha dala se \chord{C}do gala,\\
\chord{D}večerní vánek se \chord{D}do vlasů \chord{G}vplétá,\\
\chord{G}a po tý obloze \chord{G}na křídle \chord{C}rogala\\
\chord{D}s tím vánkem ve vlasech \chord{D}Markéta \chord{G}létá.
}

\vers{b}{
\chord{G}Nebe je modrý \chord{G\7}jako mý \chord{C}džíny,\\
\chord{D}tak jsme si zpívali \chord{D}s klukama \chord{G}zamlada,\\
\chord{G}zmizely smutky a \chord{G\7}podzimní \chord{C}splíny,\\
\chord{D}prostě to všechno, co \chord{D}Markéta \chord{G}nerada.
}

\vers{a}{
Vysoko na nebe, hluboko do polí\\
Markéta létá, a přitom si zpívá.\\
Co oči nevidí, to srdce nebolí,\\
je totiž podzim a brzo se stmívá.
}

\vers{b}{
Zmizely splíny a přívaly pláče\\
a s nima ty protivný přízraky z minula.\\
Připravte obvazy, dlahy a fáče,\\
kdyby se náhodou se zemí minula.
}
